All preprocessed experiments were conducted using the same hyperparameter configuration as the corresponding raw-data replications in order to isolate the effect of preprocessing. Training was performed for 300 epochs using the Adam optimizer with a learning rate of 0.005 and a weight decay of 0.01. A training batch size of 128 and an evaluation batch size of 32 were used throughout all experiments. The datasets were generated using the repository's preprocess.py script, which converted the original I-RAVEN files into serialized .pkl files containing symbolic representations, labels, and metadata. Training therefore operated directly on the preprocessed symbolic representations rather than repeatedly loading and parsing the original dataset files during execution. All experiments were executed using the adapted Python 3 and modern PyTorch implementation developed during the replication process.
