The center_single experiment was the first complete replication run performed on the I-RAVEN dataset using the adapted SCL implementation. The experiment was trained for 300 epochs with a batch size of 128, evaluation batch size of 32, learning rate of 0.005, and weight decay of 0.01. Training and evaluation were executed successfully on the remote workstation after extensive compatibility modifications to the original repository.

The center_single experiment on the I-RAVEN dataset completed successfully and achieved a final test accuracy of approximately 99.85% after 300 training epochs. The model converged steadily during training, eventually reaching nearly perfect performance on both the training and test sets. Training accuracy approached 100% relatively early in the process, while the validation/test accuracy stabilized at around 99.85%, indicating that the model was able to generalize very well on this specific configuration of the dataset.

The experiment also demonstrates that the adapted implementation is functioning correctly despite the substantial compatibility modifications that were required. Since the original repository was written for older Python and framework versions, several changes had to be introduced to make the code executable in a modern environment. These modifications included fixing Python 2 to Python 3 incompatibilities, replacing deprecated utilities, patching dataset generation code, and removing dependencies on unavailable jactorch components by substituting equivalent PyTorch functionality. The successful completion of training and evaluation indicates that these modifications preserved the intended behavior of the original implementation.

From a modeling perspective, the very high accuracy suggests that the center_single configuration is comparatively simple for the SCL architecture. The model appears capable of learning the underlying symbolic relations efficiently, as evidenced by the rapid reduction in loss and the near-perfect final performance. The small remaining gap between training and test accuracy indicates minimal overfitting, suggesting that the learned representations generalize robustly within this task configuration.

Another important outcome of this experiment is the validation of the full experimental pipeline. The workflow now includes successful dataset generation, index preparation, model training, validation, logging, checkpoint saving, and result storage. This establishes a reliable foundation for running the remaining I-RAVEN configurations such as distribute_four, distribute_nine, left_right, up_down, in_out, and in_distri. Consequently, the current experiment serves both as an initial replication result and as proof that the overall experimental environment is operational.
